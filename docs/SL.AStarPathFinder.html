Class: AStarPathFinder

SL.AStarPathFinder(start, goal, depthConstraint, neighborProviderFactory, hProvider)

Implementation of A* path finding algorithm.

The algorithm will find an optimal path between a starting node and a goal node, if one exists.

If a path is found, a sequence of AStarNodes will be returned representing the path from the start node to the goal node.

You must provide a INeighborProviderFactory that creates INeighborProviders for each examined AStarNode. You must also provide an IHeuristicProvider that gives heuristic cost estimates of distance between AStarNodes.
