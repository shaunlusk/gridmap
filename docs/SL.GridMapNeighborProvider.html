Class: GridMapNeighborProvider

SL.GridMapNeighborProvider(parent, map)

Provides neighbors for a given parent node for a given map. Returns AStarNodes for coordinates neighboring the parent node's coordinates. If allowDiagonalMovement on the provided map is false, then only coordinates primary cardinal directions are returned (N,E,S,W). Will not return neighbors for coordinates that are out of bounds or not free.
